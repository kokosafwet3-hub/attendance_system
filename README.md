# 📌 تشغيل مشروع نظام الحضور

## 🧰 المتطلبات
- Python مثبت على الجهاز

---

## 🚀 خطوات التشغيل

### 1) تثبيت المكتبات
افتح Command Prompt داخل فولدر المشروع واكتب:

pip install -r requirements.txt

---

### 2) تشغيل السيرفر

🔹 الطريقة السهلة:
اضغط دبل كليك على:
run_server.bat

🔹 أو من خلال CMD:
python manage.py runserver 0.0.0.0:8000

---

## 🌐 فتح الموقع

### من نفس الجهاز:
http://127.0.0.1:8000

---

### من الموبايل (نفس الشبكة):
1) اعرف IP الجهاز:
ipconfig

2) افتح في المتصفح:
http://192.168.x.x:8000

📌 استبدل x.x بالـ IP الحقيقي

---

## ⚠️ ملاحظات مهمة

- لازم الموبايل والكمبيوتر على نفس شبكة WiFi  
- لو الموقع لم يعمل على الموبايل:
  ✔ تأكد من فتح الـ Firewall  
- إذا تغير الـ IP:
  ✔ استخدم الـ IP الجديد فقط (لا يحتاج تعديل في المشروع)

---

## 🎯 تشغيل سريع

1) install.bat (لتثبيت المكتبات)  
2) run_server.bat (لتشغيل المشروع)
